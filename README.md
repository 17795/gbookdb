# gbookdb
 - 基于MySQL的书店数据库系统
 - 基于Flask的后端程序
 - 基于easyUI的前端网页设计

### 已完成的功能
面向客户的部分：
 - 图书信息的相关查询实现
 - 用户注册的函数
 - 登录、查询、书店主页的页面设计完成

面向管理人员的部分：
 - 网页设计完成
 - 对各个表格的添加、删除、修改、查询、通过SQL语句查询功能均已实现
 - 表格中长文本的显示方式优化
 - 修改时对空信息默认不更改
